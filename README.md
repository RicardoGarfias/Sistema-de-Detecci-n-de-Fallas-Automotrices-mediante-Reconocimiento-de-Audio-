# Sistema-de-Detecci-n-de-Fallas-Automotrices-mediante-Reconocimiento-de-Audio-
Desarrollar una app capaz de capturar el sonido de un auto, procesarlo y clasificarlo mediante un modelo de Machine Learning entrenado en Teachable Machine para determinar si corresponde a una falla.Puedes elegir mínimo 3 clases, ejemplo: Motor normal  Falla de bujía (misfire)  Golpeteo del motor (knocking) ...
